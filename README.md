## 项目介绍

本项目是一个基于 Angular 框架实现的前端甘特图组件，它主要有以下功能
1. 整体可根据按照规则配置的数据自动渲染
2. 支持树形表格数据展示
3. 支持点击滚动到任务位置
4. 支持时间轴自定义
5. 支持基本样式自定义
6. 组件的功能在持续优化完善过程中

项目主要特征如下
1. 图形区域主要使用 SVG 语法绘制，不依赖其他工具，可塑性强
2. 除 Angular 框架外，本组件无其他依赖包
3. 样式部分使用 Less 语法
